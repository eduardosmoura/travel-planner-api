# README

Simple Travel Planner with Weather API

* Ruby version
  - 2.7.1

* Rails version
  - 6.0.3.3

* System dependencies
  - RapidAPI (https://rapidapi.com/)
  - Bootstrap (https://github.com/twbs/bootstrap-rubygem#a-ruby-on-rails)

* How to run the server
  - RAPIDAPI_API_KEY=your_api_key_here rails server
